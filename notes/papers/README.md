# Paper Notes

This directory contains maintained notes for papers mentioned by the course or used to understand course topics.

Store source PDFs or source materials under `raw/papers/`. Keep the maintained reading note here.

The owner may read and write first. The agent helps explain, review, organize, and connect papers to concepts, lectures, and assignments.

## Suggested Shape

Use this shape when it helps:

```markdown
# Paper Title

## Citation

## Why This Matters

## Problem

## Core Idea

## Method

## Results

## Limitations

## Connections

## Questions

## Source
```

Do not force every section if it would be empty.

## Reading Guidance

When helping read papers:

- Start with the motivation and problem statement before equations.
- Separate what the paper claims from what later work or the course says.
- Translate unfamiliar ML notation into programming-oriented language when useful.
- Keep mathematical details, but do not let them obscure the main mechanism.
- Mark unresolved confusions rather than smoothing them over.
