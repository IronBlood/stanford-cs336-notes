# Dataset Notes

This directory contains maintained notes for datasets used or mentioned by CS336.

Keep these pages practical: what the dataset is, why it appears in the course, how it is formatted, and what questions remain before downloading or using it.

## Suggested Shape

```markdown
# Dataset Name

## What It Is

## Why It Appears In CS336

## Format And Access

## Things To Check Before Downloading

## Questions

## Sources
```

Do not download datasets just to fill out a page. Start from the dataset card and only add details after inspecting the actual files or using the dataset in code.
