# Raw Sources

This directory contains source materials for the course knowledge base.

Treat raw sources as immutable unless the owner explicitly asks otherwise.

## Layout

```text
raw/
├── papers/
├── slides/
├── transcripts/
├── assignments/
└── assets/
```

- `papers/`: PDFs or source files for papers.
- `slides/`: lecture slides or handouts.
- `transcripts/`: video transcripts and timestamped notes.
- `assignments/`: original assignment handouts or source material.
- `assets/`: images and other downloaded assets.
